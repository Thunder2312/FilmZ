A full-stack web application that allows users to browse currently showing movies, select showtimes, and book seats through an interactive UI. Built with a robust backend to handle concurrent bookings and secure user authentication.

🚀 Tech Stack

Frontend: Angular, TypeScript, HTML5, SCSS

Backend: Node.js, Express.js

Database: PostgreSQL

Authentication: JSON Web Tokens (JWT)

✨ 
Features

User Authentication: Secure login and registration with encrypted passwords.

Movie Catalog: Browse movies by genre, language, and current/upcoming status.

Interactive Seat Selection: Visual theater layout with real-time seat availability (available, selected, booked).

Booking Management: Users can view their past and upcoming ticket bookings.

Admin Dashboard: Role-based access for admins to add/remove movies, update showtimes, and manage theaters.

Responsive Design: Fully optimized for both desktop and mobile viewing.
